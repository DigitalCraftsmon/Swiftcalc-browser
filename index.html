SwiftCalc — Browser Edition

Two single-file tools. Host on GitHub Pages and run in any browser — no server needed.

Tools

Open SwiftCalc v6 Open Docs Generator

Workflow

  1. Open SwiftCalc v6 → build your estimate → Export Proposal JSON.
  2. Open Docs Generator → Load JSON, add logo & notes → Print/PDF or Save as HTML.

Tips

  • Bookmark the two pages once deployed for one-click access.
  • Saved HTML files are fully self-contained for archiving in job folders.
